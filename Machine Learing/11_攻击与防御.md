    * White Box v.s. Black Box
      * 在之前的攻击中，我们修正网络参数 $\theta$ 是为了找到可选 x'
      * 为了攻击，我们需要知道网络参数 $\theta$
        * 这被叫做白盒攻击
      * 在大多数 on-line API 中无法获取模型参数
      * 黑盒攻击是不可能的
        * 如果你有目标网络的训练资料
          * 训练一个代理网络
          * 使用这个代理网络生成攻击对象
        * 另外，从目标网络中获取输入-输出对
    * 普遍对抗性攻击 Universal Adversarial Attack
      * 对抗性攻击 Adversarial Attack
